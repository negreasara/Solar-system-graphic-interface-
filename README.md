# Solar-system-graphic-interface-
Graphic interface in Matlab showing in 3D all 8 planets, displaying the radius and surface of each, followed by a real scaled graphic with all 8 planets.
## Steps for viewing correctly the application
Firstly, make sure all files are downloaded in the same directory and folder, then open file **'MAIN_interface.m'**, hit the **RUN button** in matlab and the main page of the application will appear.
